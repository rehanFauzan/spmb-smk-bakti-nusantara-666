## Tentang SMK BAKTI NUSANTARA 666

SMK BAKTI NUSANTARA 666 adalah lembaga pendidikan kejuruan yang telah berdiri sejak tahun 1985 dan berlokasi di Jl. Percobaan, Cileunyi, Bandung. Kami berkomitmen memberikan pendidikan berkualitas tinggi dengan fasilitas modern dan tenaga pengajar profesional.

### 5 Program Keahlian Unggulan:

- **Pengembangan Perangkat Lunak dan Gim (PPLG)** - Mempersiapkan siswa menjadi programmer dan developer handal
- **Desain Komunikasi Visual (DKV)** - Mengembangkan kreativitas dalam desain grafis, branding, dan komunikasi visual
- **Animasi (ANM)** - Fokus pada pembuatan animasi 2D/3D, motion graphics, dan visual effects
- **Broadcasting dan Perfilman (BDP)** - Mempersiapkan tenaga ahli di bidang produksi media dan perfilman
- **Akuntansi dan Keuangan Lembaga (AKT)** - Mempersiapkan tenaga ahli di bidang keuangan dan akuntansi

## Instalasi dan Setup

1. Clone repository ini
```bash
git clone https://github.com/username/spmb-smk-bakti-nusantara-666.git
```

2. Install dependencies
```bash
composer install
npm install
```

3. Setup environment
```bash
cp .env.example .env
php artisan key:generate
```

4. Setup database
```bash
php artisan migrate
php artisan db:seed
```

5. Jalankan aplikasi
```bash
php artisan serve
npm run dev
```
